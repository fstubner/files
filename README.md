# AI File Explorer

A modern, AI-powered file management application built with Vue 3, TypeScript, and Tauri with Rust backend.

## Features

- Fast file navigation with Vue 3 and TypeScript
- Powerful search with Tantivy full-text indexing
- Real-time file operations tracking
- Beautiful responsive UI
- Cross-platform support (Windows, macOS, Linux)

## Tech Stack

### Frontend
- Vue 3
- TypeScript
- Pinia for state management
- Vite for bundling
- Tauri for desktop integration

### Backend
- Rust with Tauri
- Tokio for async runtime
- Tantivy for full-text search
- Serde for serialization

## Development

